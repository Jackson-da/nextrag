# 知识库检索功能

**日期**：2026-04-27  
**状态**：✅ 已完成

---

## 1. 需求背景

### 1.1 问题

- 全局检索，无法指定特定知识库
- 返回内容可能来自无关知识库

### 1.2 目标

```
用户提问 + 指定知识库 → 只检索该知识库文档 → 精准答案
```

---

## 2. 技术方案

### 方案：Metadata 过滤（推荐）

```
ChromaDB 单一 collection
       ↓
通过 filter={"kb_id": "xxx"} 过滤
       ↓
实现按知识库隔离检索
```

### 检索逻辑

```python
def get_retriever(kb_id: str | None):
    if kb_id:
        # 按知识库过滤
        return vectorstore.as_retriever(
            search_kwargs={"k": 5, "filter": {"kb_id": kb_id}}
        )
    else:
        # 全局检索
        return vectorstore.as_retriever(search_kwargs={"k": 5})
```

---

## 3. 接口设计

### 请求格式

```python
class ChatRequest:
    question: str
    session_id: str
    knowledge_base_id: str | None = None  # 指定则按知识库检索
    stream: bool = True
```

### API 示例

```bash
# 全局检索
POST /api/v1/chat/chat
{ "question": "产品功能？", "session_id": "sess-123" }

# 指定知识库检索
POST /api/v1/chat/chat
{
    "question": "定价策略？",
    "session_id": "sess-123",
    "knowledge_base_id": "kb-456"
}
```

---

## 4. 边界处理

| 场景 | 处理 |
|------|------|
| kb_id 为空 | 全局检索 |
| kb_id 不存在 | 返回空结果 |
| kb_id 无文档 | 返回空结果 |
