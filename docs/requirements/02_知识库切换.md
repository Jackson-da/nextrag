# 聊天页面知识库切换功能

**日期**：2026-04-27  
**状态**：✅ 已完成

---

## 1. 需求背景

用户在聊天页面需要能够：
- 切换到指定知识库进行精准检索
- 切换回全局检索模式
- 页面清晰显示当前检索模式

---

## 2. 功能需求

| 需求 | 优先级 | 描述 |
|------|:------:|------|
| F-001 | P0 | 支持切换知识库范围 |
| F-002 | P0 | 支持切换到"全部知识库" |
| F-003 | P0 | 切换后新消息使用新范围 |
| F-004 | P1 | 页面显示当前检索模式 |

---

## 3. UI 设计


## 4. 实现方案

### 前端

```typescript
// store/chat.ts
const currentKnowledgeBaseId = ref<string | null>(null)

function setKnowledgeBase(kbId: string | null) {
    currentKnowledgeBaseId.value = kbId
}
```

### API 参数

```typescript
// 发送消息时
const params = {
    question,
    session_id: sessionId,
    ...(currentKnowledgeBaseId.value && {
        knowledge_base_id: currentKnowledgeBaseId.value
    })
}
```
