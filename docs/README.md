# 智能问答系统文档

## 项目概述

基于 FastAPI + RAG 的智能问答系统，支持多知识库管理和用户认证。

```

    ├── 01_知识库检索.md    # ✅ 已完成
    ├── 02_知识库切换.md    # ✅ 已完成
```

## 核心功能

| 功能 | 状态 | 说明 |
|------|------|------|
| 知识库检索 | ✅ 已完成 | RAG 检索增强生成 |
| 知识库切换 | ✅ 已完成 | 支持多知识库管理 |
| 日志管理 | ✅ 已完成 | 统一日志、请求追踪、日志轮转 |
| 用户认证 | ✅ 已完成 | JWT Token 认证 |
| 对话历史管理 | ✅ 已完成 | 会话持久化、消息管理、用户隔离 |
| Redis 缓存 | ✅ 已完成 | LLM 回答缓存、多租户隔离、缓存管理 API |

## 技术栈

- **后端**: FastAPI
- **数据库**: SQLite + SQLAlchemy
- **向量库**: Chroma
- **缓存**: Redis
- **嵌入模型**: BGE-large-zh
- **大模型**: DeepSeek-chat

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
uvicorn main:app --reload
```
